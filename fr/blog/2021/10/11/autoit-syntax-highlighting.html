AutoIt coloration syntaxique dans Jetbrains

October 11, 2021

Procédure pour activer la coloration syntaxique d’AutoIt dans l’IDE Jetbrains.

Activer la coloration syntaxique dans IDE Jetbrains

  1. Ouvrir la fenetre de configuration de l’éditeur : File > Settings > Editor > TextMate Bundles ;

  2. Ajouter AutoIt bundle for Sublime Text editor en cliquant sur le bouton “plus”. Le fichier se trouve dans le répertoire de votre installation d’AutoIt C:\Program Files (x86)\AutoIt3\Extras\Editors\Sublime Text.

  3. Et voila !
